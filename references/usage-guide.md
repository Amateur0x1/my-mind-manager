# my-mind 使用指南

my-mind 是一个帮助你管理灵感、文章和创作素材的个人知识管理系统。

---

## 目录结构

```
my-mind/
├── ideas/              # 灵感库
│   ├── fleeting/       # 瞬时灵感（碎片想法）
│   ├── structured/    # 整理后的灵感
│   └── projects/      # 项目构想
├── articles/          # 文章库
│   ├── drafts/        # 草稿
│   ├── published/    # 已发布
│   └── ideas/        # 文章构想
├── assets/            # 创作素材
│   ├── images/        # 图片
│   ├── audios/        # 音频
│   ├── videos/        # 视频
│   └── attachments/  # 附件
├── inbox/             # 收集箱
│   ├── temp/          # 临时文件
│   └── to-process/    # 待处理
└── archive/           # 归档
```

---

## 快速使用

### 1. 记录灵感

```
ideas/fleeting/2026-02-26-我的想法.md
```

### 2. 写作文章

```
articles/drafts/我的文章-draft.md
```

### 3. 发布

```
articles/published/2026-02-26-我的文章.md
```
