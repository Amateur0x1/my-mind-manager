# 赛博朋克 PPT 模板

将 Markdown 转换为赛博朋克风格的幻灯片。

## 格式规则

```
# 标题 = 新幻灯片
## 二级标题 = 副标题
其他内容 = 内容
```

## 内容密度限制

| 幻灯片类型 | 最大内容 |
|-----------|---------|
| 标题页 | 1 标题 + 1 副标题 |
| 内容页 | 1 标题 + 4-6 个列表项 |

**内容太多？请拆分到多张幻灯片。禁止滚动。**

## 示例

```markdown
# 未来已来

## 技术革命

赛博时代的核心技术...

## 核心要点

- AI 生成
- 自动化
- 智能化
```

## 视觉特点

- 霓虹紫 (#ff00ff) + 青色 (#00fff2)
- 深色背景
- 发光效果文字
- 科技感字体
