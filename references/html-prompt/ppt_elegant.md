# 暗雅 PPT 模板

将 Markdown 转换为暗雅风格的幻灯片。

## 格式规则

```
# 标题 = 新幻灯片
## 二级标题 = 副标题
其他内容 = 内容
```

## 内容密度限制

| 幻灯片类型 | 最大内容 |
|-----------|---------|
| 标题页 | 1 标题 + 1 副标题 |
| 内容页 | 1 标题 + 4-6 个列表项 |

**内容太多？请拆分到多张幻灯片。禁止滚动。**

## 示例

```markdown
# 设计哲学

## 美的追求

- 简约
- 优雅
- 永恒
```

## 视觉特点

- 深灰/黑色背景
- 渐变标题
- 衬线字体
- 优雅高端
