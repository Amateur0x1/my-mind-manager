# 模板 - 纸墨文章

适合长篇阅读的纸墨风格 HTML 页面（非幻灯片）。

## 格式规则

```
# 标题 = 文章标题
## 二级标题 = 章节标题
### 三级标题 = 小节标题
其他内容 = 正文内容
```

## 特点

- 纸质背景
- 书页布局
- 衬线字体
- 适合长文章

## 适用场景

- 博客文章
- 文档说明
- 长篇内容

## 字体

- 标题：Noto Serif SC (700)
- 正文：Noto Sans SC (400)

## 颜色

```css
:root {
    --bg-primary: #f5f2eb;
    --bg-paper: #ffffff;
    --text-primary: #2c2c2c;
    --text-secondary: #666666;
    --accent: #d4af37;
}
```

## 特色元素

- 纸张质感
- 书页阴影
- 居中布局
- 衬线字体
